# Snappy mediaserver

A multiroom media server packaged as a .snap for Ubuntu Core, it consists of:

- Snapcast server
- Shairport-sync
- Mopidy

## How to install

This .snap is available in the store for release series 16 (e.g. Ubuntu
16.04). Install via:

    $ sudo snap install mediaserver-frafall
